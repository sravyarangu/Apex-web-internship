📚 Portfolio Setup Guide

Step 1: Verify XAMPP Installation

Ensure XAMPP is installed with Apache and MySQL running.

C:\xampp\apache\bin\httpd.exe -v # Check Apache

✓ Apache running on Port 80

✓ MySQL running on Port 3306

Step 2: Import Database Schema

Create the portfolio database using phpMyAdmin or MySQL CLI.

mysql -u root -p < database/portfolio_db.sql

✓ Database "portfolio_db" created

✓ Tables: contact_messages, projects, skills

Step 3: Start the Website

Open your browser and navigate to:

http://localhost/portfolio/

✓ Website loads successfully

✓ All CSS and JavaScript files load

Step 4: Test Contact Form

Fill out the contact form to test PHP and MySQL integration.

Form validation → PHP processing → Database storage

✓ Message saved in contact_messages table

✓ Success confirmation displayed

Step 5: View Messages in phpMyAdmin

http://localhost/phpmyadmin → portfolio_db → contact_messages

✓ Messages visible in database
